# NaturGuard - Natural Mosquito Repellent Website

## Overview
A professional, fully functional website for NaturGuard mosquito repellent product made from natural neem leaves and ingredients.

## Files Included
- `index.html` - Main HTML file with all sections
- `styles.css` - Complete styling with green color scheme
- `script.js` - JavaScript for navigation and interactions
- `team-photo.jpg` - Team photo image
- `product-photo.jpg` - Product photo image

## Features
✅ Fully responsive design (works on desktop, tablet, and mobile)
✅ Smooth scrolling navigation
✅ Animated sections and elements
✅ Interactive hover effects
✅ Professional green color scheme (dark green, green, light green, white, black, grey)
✅ Multiple pages/sections:
   - Home (with team photo)
   - Ingredients (chemicals used)
   - Procedure (step-by-step manufacturing)
   - Process Flow Diagram (visual flowchart)
   - Media Gallery
   - Applications (uses and benefits)

## How to Use

### Option 1: Open Directly
1. Extract all files to a folder
2. Double-click `index.html` to open in your browser

### Option 2: Use a Local Server (Recommended)
1. Install Python (if not already installed)
2. Open terminal/command prompt in the folder
3. Run: `python -m http.server 8000`
4. Open browser and go to: `http://localhost:8000`

### Option 3: Deploy Online
You can host this website for free on:
- GitHub Pages
- Netlify
- Vercel
- Any web hosting service

## Navigation
Click on any menu item to jump to that section:
- **Home** - Hero section with team photo and stats
- **Ingredients** - Details about neem leaves, coconut oil, menthol crystals, and odor neutralizer
- **Procedure** - 8-step manufacturing process
- **Process Flow** - Visual diagram of the production process
- **Media** - Product gallery and images
- **Applications** - Different uses and how to use the product

## Customization
To modify content:
1. Open `index.html` in any text editor
2. Edit text within the HTML sections
3. Save and refresh your browser

To modify colors:
1. Open `styles.css`
2. Change the color values in the `:root` section at the top
3. Save and refresh

## Technical Details
- Pure HTML5, CSS3, JavaScript (no external frameworks needed)
- Google Fonts: Playfair Display & Outfit
- Responsive design with CSS Grid and Flexbox
- Smooth scroll animations
- Intersection Observer for scroll animations
- SVG for process flow diagram
